# Aria Sidequest

A recovery-first Year 7 learning hub with short visual sidequests, saved progress, and parent-controlled limits.

**Live app:** https://zbpftdjswk-creator.github.io/aria-sidequest/

## What is included

- 27 bite-sized missions across Maths, English, Science, French, Geography, History, and Computing
- seven cinematic featured episodes that rotate at local midnight
- visual, one-tap activities with finite sessions and no autoplay
- optional gentle sounds for opening an episode and celebrating success
- sound is off by default, has no negative wrong-answer noise, and can be muted at any time
- local progress, XP, rosettes, daily picks, and a 20-minute parent-controlled cap
- responsive layouts for iPad, mobile, and desktop

## Data and privacy

- Assessment reports and diagnostic information are **not** included in this repository.
- Progress and the sound preference are stored only in the browser using local storage.
- The app has no account system, analytics, advertising, or data-upload feature.
- The live GitHub Pages site is public, but it asks search engines not to index it.

## Run locally

Download `index.html` and open it in a modern browser. An internet connection is used only to load the interface icon library; learning progress stays on the device.
