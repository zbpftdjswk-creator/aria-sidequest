# Aria Sidequest

A private, recovery-first Year 7 learning hub with short visual sidequests, saved progress, and parent-controlled limits.

## What is included

- 27 bite-sized missions across Maths, English, Science, French, Geography, History, and Computing
- visual, one-tap activities with finite sessions and no autoplay
- local progress, XP, rosettes, daily picks, and a 20-minute parent-controlled cap
- responsive layouts for iPad, mobile, and desktop

## Data and privacy

- Assessment reports and diagnostic information are **not** included in this repository.
- Progress is stored only in the browser using local storage.
- The app has no account system, analytics, advertising, or data-upload feature.
- GitHub Pages sites are normally public even when their source repository is private, so hosting should only be enabled after reviewing that visibility.

## Run privately

Download `index.html` and open it in a modern browser. An internet connection is used only to load the interface icon library; learning progress stays on the device.
